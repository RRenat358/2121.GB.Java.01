
#Описание и правила игры.

По очереди с компьютером ставим X или 0.
Для постановки своего символа == X:
-- указываем координату по X и по Y -- по очереди.

Если указать на занятое поле -- предлагается ввести ещё раз.

Выигрывает тот, кто поставил N-количество символов подряд:
-- по горизонтали
-- по вертикале
-- по диагонали квадрата (из угла в угол)

Можно указать:
Размер поля (указать в коде) == 3 по умолчанию.
N-количество символов подряд (указать в коде) == 3 по умолчанию.
